## Lab04: AI Agent with LangGraph

* **Core:** Python 3.10+
* **Orchestration:** LangGraph (State Machine)
* **LLM:** GPT-4o-mini (OpenAI)
* **Interface:** LangChain Core & Prebuilt ToolNode
* **Environment:** Python-dotenv

## 📁 Folder Stucture

```text
lab4_agent/
├── agent.py           # Luồng điều khiển chính và cấu hình Graph
├── tools.py           # Định nghĩa các công cụ: Tìm bay, khách sạn, tính tiền
├── system_prompt.txt  # "Bộ não" định hình tính cách và quy tắc của Agent
├── log_chat.txt       # Nhật ký hoạt động của Agent (tự động sinh ra)
├── .env               # Chứa OPENAI_API_KEY (không push lên GitHub)
└── requirements.txt   # Danh sách thư viện cần thiết
```
